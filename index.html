untemplate.js

This module helps you scrape structured information from websites with HTML templates. Read more →

Templating engines like handlebars/jade:

{ structured: 'info' } + {{ template }} = <html>

This module:

<html> - {{ template }} = { structured: 'info' }

This tool provides an interface for the #deduceTemplate function. Copy and paste HTML snippets of components you want to scrape in the field below. Add a handful of diverse examples to get a robust template.
